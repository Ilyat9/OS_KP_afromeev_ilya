# Особенности реализации
- **IPC:** Использование `mkfifo`, `read`, `write` для обмена сообщениями.
- **Мультиплеер:** Поддержка нескольких игроков в одной игровой сессии (лобби).
- **Функционал:** - Создание игры с указанием названия и лимита игроков.
  - Подключение к конкретной игре по имени.
  - Поиск случайной свободной игры (Matchmaking).
  - Подсчет "быков" и "коров" на стороне сервера.

# Структура проекта
- `server.c` — Логика сервера: управление сессиями, генерация чисел, арбитраж игры.
- `client.c` — Клиентское приложение: интерфейс пользователя и игровой цикл.
- `defs.h` — Общие структуры данных и протокол обмена.
- `Makefile` — Скрипт автоматизации сборки.

# Инструкция по сборке и запуску

## 1. Сборка
Для компиляции сервера и клиента выполните команду:
```bash
make
```

# Запуск сервера
Запустите сервер в первом терминале:
```bash
./server
```
# Запуск клиента
В новых терминалах запустите одного или нескольких клиентов:

```Bash
./client
```
# Игровой процесс
Один клиент выбирает "Create Game", вводит имя и количество игроков (например, 2).

Другой клиент выбирает "Find Random Game" или "Join by Name".

После заполнения лобби сервер присылает уведомление о начале игры.

Игроки по очереди вводят 4-значные числа, пока кто-то не угадает секретную комбинацию.

# Очистка
Чтобы удалить исполняемые файлы и временные FIFO:

```Bash

make clean
```
